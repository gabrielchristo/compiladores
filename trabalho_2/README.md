## Trabalho 2 - Geração de forma intermediária

### Gramática utilizada

atribuição
A -> id { Print(id); } = E { Print("="); } | P

operadores (com precedência)
E -> E + T { Print("+"); }
   | E - T { Print("-"); }
   | T
T -> T * F { Print("*"); }
   | T / F { Print("/"); }
   | F
   
terminais
F -> id { Print(id + "@"); }
   | num { Print(num); }
   | E

### Eliminando recursividade à esquerda
A -> A a | ß, Se torna:

A -> ß A'
A' -> a A' | e

### Gramática sem recursividade à esquerda

A -> id { Print( id ); } = E { Print( "="); }
E -> T E'
E' -> + T { Print( "+"); } E'
    | - T { Print( "-"); } E'
    | e
T -> F T'
T' -> * F { Print( "*"); } T'
    | / F { Print( "/"); } T'
    | e
F -> id { Print(id + "@"); }
   | num { Print( num ); }
   | ( E )
